# Mini Project: Appointment Booking System

Project Description:
The Appointment Booking System is a Java Full Stack web application developed using JSP, JDBC, MySQL, HTML, CSS, and Apache Tomcat. It allows users to book appointments online and stores the appointment details in a MySQL database.

Features:
- Book appointments online
- Store appointment details in MySQL
- Simple and user-friendly interface
- Displays booking confirmation

Technologies Used:
- Java (JSP)
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat
- Eclipse IDE

Project Structure:
Mini-Project
│
├── PPT
├── Documentation
├── Source_Code
├── Database
├── Screenshots

Database:
- Database Name: 'appointmentdb'
- Table Name: 'appointments'

How to Run:
1. Import the project into Eclipse.
2. Configure Apache Tomcat Server.
3. Create the MySQL database using "appointmentdb.sql".
4. Add the MySQL Connector JAR.
5. Run the project on Tomcat.
6. Open the application in your browser.

Output:
The application stores appointment details in the MySQL database and displays a successful booking message.
